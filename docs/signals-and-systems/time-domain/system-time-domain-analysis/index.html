System Time Domain Analysis

When we say time domain, we mean that we look at the signal with its original form. The variable may not necessarily be time. It can also be space or even frequency. It is just how we are analyzing the signal.

Definitions​

Casual System​

Casual System is a system that does not entail the future state of the signal.

For example,

S(x(t))=x(t)+x(t−1)\mathcal{S}(x(t)) = x(t) + x(t-1)S(x(t))=x(t)+x(t−1)

is casual, whereas,

S(x(t))=x(t)+x(t+1)\mathcal{S}(x(t)) = x(t) + x(t+1)S(x(t))=x(t)+x(t+1)

Is not.

tip

When we use time as the variable, it may seem a bit strange. But non-casual system is common in cases like image processing.

Linear Systems​

A linear system is a system that satisfy,

S(αx+βy)=αS(x)+βS(y)\mathcal{S}(\alpha x + \beta y) = \alpha \mathcal{S}(x) + \beta \mathcal{S}(y)S(αx+βy)=αS(x)+βS(y)

Time Invariant Systems​

A time invariant system is a system that satisfy,

S(x(τ−a))=S(x(t))∣t=τ−a\mathcal{S}(x(\tau-a)) = \mathcal{S}(x(t))|_{t=\tau-a}S(x(τ−a))=S(x(t))∣t=τ−a​

LTI​

We almost only focus on LTI, linear and time invariant systems.

Typical LTI Systems​

  • Delay or Shift S(x(t))=x(t−a)\mathcal{S}(x(t)) = x(t-a)S(x(t))=x(t−a)
  • Scale or Amplify S(x(t))=αx(t)\mathcal{S}(x(t)) = \alpha x(t)S(x(t))=αx(t)
  • Integration S(x(t))=∫−∞tx(τ)dτ\mathcal{S}(x(t)) = \int_{-\infty}^t x(\tau) d\tauS(x(t))=∫−∞t​x(τ)dτ
  • Differentiation S(x(t))=ddtx(t)\mathcal{S}(x(t)) = \frac{\mathrm{d}}{\mathrm{d}t} x(t)S(x(t))=dtd​x(t)

System Time Domain Analysis​

Convolution​

Convolution is a very important operation in time domain analysis.

Defined as,

x(t)∗y(t)=∫−∞+∞x(τ)y(t−τ)dτx(t) \ast y(t) = \int_{-\infty}^{+\infty} x(\tau) y(t - \tau) d\taux(t)∗y(t)=∫−∞+∞​x(τ)y(t−τ)dτ

Convolution is a linear operation to both of the signals.

It is also symmetric for both of the signals.

It is also associative.

The above three properties are obvious.

Convolution is a LTI System​

Consider a system,

S(x(t))=x(t)∗y(t)\mathcal{S}(x(t)) = x(t) \ast y(t)S(x(t))=x(t)∗y(t)

because convolution is linear, the system is also linear. You can also prove that the system is time invariant.

Impulse Response​

We define,

S(δ(t))=h(t)\mathcal{S}(\delta(t)) = h(t)S(δ(t))=h(t)

As the impulse response of the system.

This is important because, let's consider convolution for δ(t)\delta(t)δ(t),

δ(t)∗y(t)=∫−∞+∞δ(τ)y(t−τ)dτ=y(t)\delta(t) \ast y(t) = \int_{-\infty}^{+\infty} \delta(\tau) y(t - \tau) d\tau = y(t)δ(t)∗y(t)=∫−∞+∞​δ(τ)y(t−τ)dτ=y(t)

And because convolution is linear, so,

S(δ(t))∗y(t)=h(t)∗y(t)\mathcal{S}(\delta(t)) \ast y(t) = h(t) \ast y(t)S(δ(t))∗y(t)=h(t)∗y(t)

This is why h(t)h(t)h(t) is also called the system response function in time domain.

The above equation tells us that every LTI system has a unique h(t)h(t)h(t) that can fully describe the behavior of the system.

  • Scale or Amplify S(x(t))=αx(t)\mathcal{S}(x(t)) = \alpha x(t)S(x(t))=αx(t), h(t)=αδ(t)h(t) = \alpha \delta(t)h(t)=αδ(t)
