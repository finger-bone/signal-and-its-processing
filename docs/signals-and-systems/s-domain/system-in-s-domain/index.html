System in S Domain

We will say how Laplace transform solves the issue of jump value. But before that, we need some trivial work done.

System Response Function in S Domain​

Because fourier transform and laplace transform follows the same rule of convolution, and LTI system can always be described by the system impulse response, that is,

y(t)=x(t)∗h(t)y(t) = x(t) \ast h(t)y(t)=x(t)∗h(t)

Just like how we did with fourier transformation, we can use,

Y(s)=X(s)H(s)Y(s) = X(s)H(s)Y(s)=X(s)H(s)

H(s)H(s)H(s) is the system response function in s domain.

Solving System in Differential Equation by Laplace Transform​

To describe this part in a more general way is hard. So let's only look at a system described by a differential equation of the second order. What we are introducing is valid for higher orders, it's just more complex.

(aD2+bD+c)y=x(aD^2+bD+c)y = x(aD2+bD+c)y=x

Now let's apply laplace transform. Please note that, for laplace transform, the derivative will result in an extra term.

Since,

Df→LsL(s)−f(−0)Df \xrightarrow{\mathcal{L}} s L(s) - f(-0)DfL​sL(s)−f(−0)

Thus,

D2f=DDf→Ls(sL(s)−f(−0))−f′(−0)=s2L(s)−sf(−0)−f′(−0)D^2f = DDf \\ \xrightarrow{\mathcal{L}} s(sL(s) - f(-0)) - f'(-0) \\ = s^2 L(s) - sf(-0) - f'(-0)D2f=DDfL​s(sL(s)−f(−0))−f′(−0)=s2L(s)−sf(−0)−f′(−0)

Put this into the original equation,

as2Y(s)−asf(−0)−af′(−0)+bsY(s)−bf(−0)+cY(s)=X(s)as^2 Y(s) - asf(-0) - af'(-0) + b s Y(s) - bf(-0) + c Y(s) = X(s)as2Y(s)−asf(−0)−af′(−0)+bsY(s)−bf(−0)+cY(s)=X(s)

Let's group them,

(as2+bs+c)Y(s)=X(s)+asf(−0)+af′(−0)+bf(−0)(as^2 + bs + c) Y(s) = X(s) + asf(-0) + af'(-0) + bf(-0)(as2+bs+c)Y(s)=X(s)+asf(−0)+af′(−0)+bf(−0)

We can tell that the right hand side is made up of two parts,

X(s)X(s)X(s) is the external input. Where as asf(−0)+a′f(−0)+bf(−0)asf(-0) + a'f(-0) + bf(-0)asf(−0)+a′f(−0)+bf(−0) is caused by internal state.

We treat both the external input and the internal state as the input of the system, so we can write,

H(s)=1as2+bs+cH(s) = \frac{1}{as^2 +bs+c}H(s)=as2+bs+c1​

Due to the linearity, the solution is also in two parts,

Y0s(s)=X(s)as2+bs+cY0i(s)=asf(−0)+af′(−0)+bf(−0)as2+bs+cY_{0s}(s) = \frac{X(s)}{as^2 +bs+c} \\ Y_{0i}(s) = \frac{asf(-0) + af'(-0) + bf(-0)}{as^2 +bs+c}Y0s​(s)=as2+bs+cX(s)​Y0i​(s)=as2+bs+casf(−0)+af′(−0)+bf(−0)​

The Y0s(s)Y_{0s}(s)Y0s​(s) is the zero state response. Because it presumes f(n)(−0)=0f^{(n)}(-0) = 0f(n)(−0)=0.

The Y0i(s)Y_{0i}(s)Y0i​(s) is the zero input response. Because it presumes Y(s)=0Y(s) = 0Y(s)=0.

The final Y(s)=Y0s(s)+Y0i(s)Y(s) = Y_{0s}(s) + Y_{0i}(s)Y(s)=Y0s​(s)+Y0i​(s). To get the time domain result, we simply perform a reverse laplace transform.

Please note that this solution is only dependent on f(n)(−0)f^{(n)}(-0)f(n)(−0). And thus there is no need to calculate the jump value.

In addition, please note the fact that,

Y(s)=Y0s(s)+Y0i(s)Y(s) = Y_{0s}(s) + Y_{0i}(s)Y(s)=Y0s​(s)+Y0i​(s)

Is only valid for t>−0t > -0t>−0. We can't have any information for t<−0t < -0t<−0 because we use single side laplace transform. If we want the convergence of the single side, we lose half of the solution.

info

You can also do this separation in the time domain. It is just that it's not obvious.

For a differential equation,

P(D)y(t)=x(t)P(D)y(t) = x(t)P(D)y(t)=x(t)

With initial values, y(n)(−0)y^{(n)}(-0)y(n)(−0) given.

If we presume x(t)=0x(t) = 0x(t)=0, the solution is the zero input response in the time domain, which is in pair with the zero input response in s domain. And vice versa for zero state response solved by letting y(n)(−0)=0y^{(n)}(-0)=0y(n)(−0)=0 instead of the given values.

In the end, the solution is the sum of the two solutions.

y(t)=y0s(t)+y0i(t)y(t) = y_{0s}(t) + y_{0i}(t)y(t)=y0s​(t)+y0i​(t)

However, please note that, because we used single side laplace transformation, we can only solve the response from −0-0−0 to +∞+\infty+∞. So we have only y(t)y(t)y(t) on the positive side.

Example​

If we choose,

a=1b=4c=3f(−0)=1f′(−0)=1x(t)=δ(t)+u(t)a = 1 \quad b = 4 \quad c = 3 \quad f(-0) = 1 \quad f'(-0) =1 \quad x(t) = \delta(t) + u(t)a=1b=4c=3f(−0)=1f′(−0)=1x(t)=δ(t)+u(t)

We will have,

X(s)=1+1sX(s) = 1 + \frac{1}{s}X(s)=1+s1​

Thus, for the zero state response,

Y0s(s)=1+1ss2+4s+3=s+1(s+3)(s+1)s=1(s+3)(s+1)=12(1s+1−1s+3)Y_{0s}(s) = \frac{1+\frac{1}{s}}{s^2 +4s+3} \\ = \frac{s+1}{(s+3)(s+1)s} \\ = \frac{1}{(s+3)(s+1)} \\ = \frac{1}{2}(\frac{1}{s+1} - \frac{1}{s+3})Y0s​(s)=s2+4s+31+s1​​=(s+3)(s+1)ss+1​=(s+3)(s+1)1​=21​(s+11​−s+31​)

Because,

e−at→L1s+ae^{-at} \xrightarrow{\mathcal{L}} \frac{1}{s+a}e−atL​s+a1​

Thus, for the positive time,

y0s(t)=12(e−t−e−3t)y_{0s}(t) = \frac{1}{2}(e^{-t} - e^{-3t})y0s​(t)=21​(e−t−e−3t)

And for the zero input response,

Y0i=s+5s2+4s+3=s+5(s+3)(s+1)=2s+1−1s+3Y_{0i} = \frac{s+5}{s^2 + 4s + 3} \\ = \frac{s+5}{(s+3)(s+1)} \\ = \frac{2}{s+1} - \frac{1}{s+3}Y0i​=s2+4s+3s+5​=(s+3)(s+1)s+5​=s+12​−s+31​

y0i=2e−t−e−3ty_{0i} = 2 e^{-t} - e^{-3t}y0i​=2e−t−e−3t

This is why laplace transform doesn't need a jump value- we only need the information at −0-0−0.
