System in Differential Equation

Previously we introduced the properties of signals and systems in the time domain. Here we focus on a common case, where a system is defined by a differential equation and how can we solve them.

We use DDD as the differential operator for convenience.

TLI system is always defined as a linear differential equation, that is,

y(t)=S(x(t))y(t) = \mathcal{S}(x(t))y(t)=S(x(t)) P(D)y(t)=x(t)P(D)y(t) = x(t)P(D)y(t)=x(t)

Where P(D)P(D)P(D) is a polynomial of operator DDD. For convenience, we force the constant term to exists. Or else, we can always integrate the both side to get an equivalent equation with our desired P(D)P(D)P(D).

We can usually solve this easily with the following method for common inputs,

Eigen Decomposition​

We decompose the result y(t)y(t)y(t) we are looking for into two parts,

y(t)=y‾(t)+y∗(t)y(t) = \overline{y}(t) + y^*(t)y(t)=y​(t)+y∗(t)

Where,

P(D)y‾(t)=0P(D)\overline{y}(t) =0P(D)y​(t)=0

Called the eigen solution.

And,

P(D)y∗(t)=x(t)P(D)y^*(t) = x(t)P(D)y∗(t)=x(t)

Called the special solution.

Eigen Solution​

Let's first look for,

P(D)y‾(t)=0P(D)\overline{y}(t) = 0P(D)y​(t)=0

If we take,

L(D)=P(D)−μL(D) = P(D) - \muL(D)=P(D)−μ

Where μ\muμ is the coefficient of the zeroth order term in PPP, then,

L(D)y‾(t)=μy‾(t)L(D)\overline{y}(t) = \mu \overline{y}(t)L(D)y​(t)=μy​(t)

That is equal to say we should look for the eigen function of L(D)L(D)L(D).

Special Solution​

Now we want,

P(D)y∗(t)=x(t)P(D)y^*(t) = x(t)P(D)y∗(t)=x(t)

We need to look for,

P(D)−1x(t)P(D)^{-1}x(t)P(D)−1x(t)

Because we can always use Taylor expansion to make,

P(D)−1=P′(D)P(D)^{-1} = P'(D)P(D)−1=P′(D)

So formally,

y(t)=P′(D)x(t)y(t) = P'(D)x(t)y(t)=P′(D)x(t)

But P′(D)P'(D)P′(D) has infinite terms, so this is actually hard to solve.

But if x(t)x(t)x(t) is only a polynomial of ttt with the highest order of nnn, we can truncate P′(D)P'(D)P′(D) to order nnn, so that it is easy to solve.

However, if there exists eate^{at}eat in x(t)x(t)x(t), truncate can't be done. But we can use the following of what we call the shifting theorem,

Because,

D(eatx(t))=(D+a)eatx(t)D(e^{at} x(t)) = (D+a)e^{at}x(t)D(eatx(t))=(D+a)eatx(t)

So we can conclude,

P(D)eatx(t)=eatP(D+a)x(t)P(D) e^{at}x(t) = e^{at}P(D+a)x(t)P(D)eatx(t)=eatP(D+a)x(t)

This is the shifting theorem.

Differential Equation with Pulse signals​

Integration Example​

The previous method is very nice. However, when come to equations like,

P(D)y(t)=∑i=0+∞AiDiδ(t)+u(t)s(t)P(D)y(t) = \sum_{i=0}^{+\infty} A_i D^{i} \delta(t) + u(t) s(t)P(D)y(t)=i=0∑+∞​Ai​Diδ(t)+u(t)s(t)

Of course, we can solve this by parts with the previous method. That is, we can solve,

P(D)y(t)=s(t)t>0P(D)y(t)=0t<0P(D)y(t) = s(t) \quad t > 0 \\ P(D)y(t) = 0 \quad t < 0P(D)y(t)=s(t)t>0P(D)y(t)=0t<0

Instead.

This seems to have solved the issue. But we will have multiple sets of solutions and undefined coefficients- but according to the theory, we should only need one initial state.

We illustrate this by,

Dy+y=δ(t)+u(t)Dy + y = \delta(t) + u(t)Dy+y=δ(t)+u(t)

For t>0t > 0t>0,

y=Ae−t+1y = Ae^{-t} + 1y=Ae−t+1

For t<0t < 0t<0,

y=(B−A)e−ty = (B-A)e^{-t}y=(B−A)e−t

However, since this equation is one order, the following initial state,

y(−0)=4y(-0) = 4y(−0)=4

Should give us a complete solution- but we have two coefficients here.

This is because, AAA and BBB are not independent of each other.

We can rewrite the above solution into one with step function,

y(t)=(A+Bu(t))e−t+u(t)y(t) = (A + Bu(t))e^{-t} + u(t)y(t)=(A+Bu(t))e−t+u(t)

Let's consider have the differential equation behaves around 000.

Dy+y=δ(t)+u(t)Dy + y = \delta(t) + u(t)Dy+y=δ(t)+u(t)

We integrate on the both sides from −ϵ-\epsilon−ϵ to ϵ\epsilonϵ,

∫−ϵϵ(Dy+y)dt=∫−ϵϵ(δ(t)+u(t))dt\int_{-\epsilon}^{\epsilon} (Dy+y) \mathrm{d}t = \int_{-\epsilon}^{\epsilon} (\delta(t) + u(t)) \mathrm{d}t∫−ϵϵ​(Dy+y)dt=∫−ϵϵ​(δ(t)+u(t))dt y(ϵ)−y(−ϵ)+∫−ϵϵy(t)dt=∫−ϵϵ(δ(t)+u(t))dty(\epsilon) - y(-\epsilon) + \int_{-\epsilon}^{\epsilon} y(t) \mathrm{d}t = \int_{-\epsilon}^{\epsilon} (\delta(t) + u(t)) \mathrm{d}ty(ϵ)−y(−ϵ)+∫−ϵϵ​y(t)dt=∫−ϵϵ​(δ(t)+u(t))dt

We make ϵ→0\epsilon \to 0ϵ→0, so,

y(+0)−y(−0)=1y(+0) - y(-0) = 1y(+0)−y(−0)=1

That is to say,

y(−0)=4y(+0)=5y(-0) = 4 \quad y(+0) = 5y(−0)=4y(+0)=5

Now we can have,

A=4A+B+1=5A = 4 \quad A+B+1 = 5A=4A+B+1=5

Now we can solve AAA and BBB.

This is how we can solve differential equation with pulse signals.

Similarly, for equations of higher order, there will be y(n)(+0)−y(n)(−0)y^{(n)}(+0) - y^{(n)}(-0)y(n)(+0)−y(n)(−0) as well.

You can always analyze by integration, the method above is too grinding- we have a better trick called pulse balancing.

Pulse Balancing​

Let's consider the differential equation,

P(D)y(t)=∑i=0+∞AiDiδ(t)+Cu(t)+s(t)P(D)y(t) = \sum_{i=0}^{+\infty} A_i D^{i} \delta(t) + Cu(t) + s(t)P(D)y(t)=i=0∑+∞​Ai​Diδ(t)+Cu(t)+s(t)

What results in the jump value? It is the discontinuous part of the right side, that is, all δ(t)\delta(t)δ(t) related function and Cu(t)Cu(t)Cu(t).

tip

Why discontinuous part on the right side causes jump value? We can integrate on the both sides, just like before.

We can always consider moving the right side to into the y(t)y(t)y(t), that is,

P(D)(y(t)−∑i=0+∞Ai′Diδ(t)−C′u(t))=s(t)P(D)(y(t) - \sum_{i=0}^{+\infty} A'_i D^{i} \delta(t) - C'u(t)) = s(t)P(D)(y(t)−i=0∑+∞​Ai′​Diδ(t)−C′u(t))=s(t)

If s(t)s(t)s(t) is continuous, then y(t)−∑i=0+∞Ai′Diδ(t)−C′u(t)y(t) - \sum_{i=0}^{+\infty} A'_i D^{i} \delta(t) - C'u(t)y(t)−∑i=0+∞​Ai′​Diδ(t)−C′u(t) is continuous. Because the ∑i=0+∞Ai′Diδ(t)+C′u(t)\sum_{i=0}^{+\infty} A'_i D^{i} \delta(t) + C'u(t)∑i=0+∞​Ai′​Diδ(t)+C′u(t) part is discontinuous, and so is y(t)y(t)y(t), their jump value must be the same so that they can be deduced. So we can just integrate ∑i=0+∞Ai′Diδ(t)+C′u(t)\sum_{i=0}^{+\infty} A'_i D^{i} \delta(t) + C'u(t)∑i=0+∞​Ai′​Diδ(t)+C′u(t) for the jump value- and better off, the integration is easy. For the zeroth order, it is obvious C′C'C′, for the iii th order, it is, (i)!Ai−1′(i)!A'_{i-1}(i)!Ai−1′​.

This is kind of abstract- let's demonstrate with an example,
